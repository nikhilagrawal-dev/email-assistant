# Smart Email Assistant

A full-stack AI-powered email assistant that generates replies directly inside Gmail.

## Features
- Chrome Extension for Gmail integration
- Spring Boot backend
- AI-based reply generation
- Tone & length selection
- History tracking

## Tech Stack
- Backend: Spring Boot (Java)
- Frontend: React
- Extension: JavaScript (Chrome Extension)

## Project Structure
- backend/ → Spring Boot API
- frontend/ → React UI
- extension/ → Chrome extension

## Status
AI integration implemented with fallback support.
